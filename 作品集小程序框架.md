# 视觉设计师个人作品集小程序框架

## 一、项目概述

这是一个为视觉设计师打造的个人作品集小程序，模仿花瓣APP的布局和操作体验，支持作品上传、分类管理、扫码查看等功能。

## 二、核心功能

### 1. 前端展示功能
- **瀑布流布局**：模仿花瓣APP的视觉效果，实现作品的瀑布流展示
- **作品分类**：支持书籍装帧、UI设计、海报设计、插画设计等多种分类
- **作品详情**：展示作品的详细信息，包括多张图片、描述、创作年份等
- **关于我**：展示设计师个人信息、联系方式和专业技能
- **扫码查看**：生成二维码，方便他人扫码访问作品集

### 2. 后台管理功能
- **管理员登录**：安全的登录验证
- **作品管理**：上传、编辑、删除作品
- **分类管理**：添加、编辑、删除分类
- **数据统计**：查看作品浏览量、总访问量等数据

## 三、技术架构

### 1. 前端技术栈
- React 19.2.4
- React Router 6.28.0
- Tailwind CSS 4.0.0
- Axios 1.7.9
- React Hook Form 7.53.2
- ColorThief 2.3.2（用于提取图片主色调）

### 2. 后端技术（可选）
- Node.js + Express
- MongoDB 或 MySQL
- JWT 认证

### 3. 部署方案
- Vercel 静态部署
- 或使用微信小程序平台

## 四、页面结构

### 1. 前台页面
- **首页**：瀑布流展示所有作品，支持分类筛选
- **作品详情页**：展示作品详细信息，支持图片轮播
- **关于我页**：个人介绍、联系方式、技能展示
- **分类页**：按分类浏览作品

### 2. 后台页面
- **登录页**：管理员登录界面
- **控制台**：数据概览、统计信息
- **作品管理**：作品列表、上传、编辑、删除
- **分类管理**：分类列表、添加、编辑、删除

## 五、核心功能实现

### 1. 瀑布流布局
- 使用 CSS Grid 或 Masonry 布局实现
- 按作品主色调排序，实现色彩过渡效果
- 响应式设计，适配不同设备

### 2. 作品上传
- 支持多图片上传
- 自动提取图片主色调
- 支持设置作品标题、描述、分类、年份等信息
- 图片压缩和优化

### 3. 分类管理
- 支持自定义分类名称
- 支持分类排序
- 统计每个分类下的作品数量

### 4. 扫码查看
- 生成作品集的二维码
- 支持分享到社交媒体
- 游客模式，无需登录即可查看

### 5. 数据统计
- 作品浏览量统计
- 总访问量统计
- 分类访问热度分析

## 六、用户权限

### 1. 游客
- 可查看所有作品
- 可浏览分类
- 可查看关于我页面
- 不可上传或编辑作品

### 2. 管理员
- 拥有所有权限
- 可上传、编辑、删除作品
- 可管理分类
- 可查看数据统计

## 七、界面设计

### 1. 设计风格
- 现代简约风格
- 突出作品本身
- 响应式设计
- 流畅的动画效果

### 2. 色彩方案
- 主色调：#3b82f6（蓝色）
- 辅助色：#f97316（橙色）
- 中性色：#f3f4f6（浅灰）、#1f2937（深灰）

### 3. 交互体验
- 平滑的滚动效果
- 图片悬停放大效果
- 分类切换动画
- 作品详情页的淡入效果
