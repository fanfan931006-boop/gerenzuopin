# 微信云开发使用指南

## 一、云开发简介

微信云开发是微信团队提供的后端云服务，无需搭建服务器，即可实现数据库、存储、云函数等功能。对于个人作品集小程序来说，云开发是一个理想的选择，因为它：

- **无需服务器**：无需购买和配置服务器
- **无需域名备案**：直接使用微信提供的云服务
- **开发效率高**：前端直接调用云API，减少前后端联调
- **免费额度**：提供基础免费额度，适合个人项目

## 二、开通云开发

1. **打开微信开发者工具**：确保已安装最新版本
2. **创建或打开项目**：选择小程序项目
3. **开通云开发**：
   - 点击左侧导航栏的「云开发」按钮
   - 点击「开通」按钮
   - 填写环境名称（如 `portfolio-env`）
   - 选择「免费版」即可（适合个人项目）

## 三、云环境配置

1. **获取环境ID**：
   - 开通云开发后，在云开发控制台首页可以看到环境ID
   - 记录下环境ID，后续代码中需要使用

2. **初始化云环境**：
   - 在小程序代码根目录的 `app.js` 文件中添加云环境初始化代码：
   ```javascript
   // app.js
   App({
     onLaunch() {
       // 初始化云开发环境
       wx.cloud.init({
         env: 'your-env-id', // 替换为你的环境ID
         traceUser: true
       })
     }
   })
   ```

## 四、云数据库设计

### 1. 集合设计

| 集合名称 | 描述 | 字段 |
|---------|------|------|
| `works` | 作品数据 | `_id` (自动生成), `title` (标题), `description` (描述), `images` (图片数组), `category` (分类ID), `year` (年份), `dominantColor` (主色调), `viewCount` (浏览量), `createdAt` (创建时间), `updatedAt` (更新时间) |
| `categories` | 分类数据 | `_id` (自动生成), `name` (分类名称), `order` (排序), `createdAt` (创建时间), `updatedAt` (更新时间) |
| `admins` | 管理员数据 | `_id` (自动生成), `username` (用户名), `password` (密码), `createdAt` (创建时间) |
| `visitStats` | 访问统计 | `_id` (自动生成), `totalVisits` (总访问量), `workVisits` (作品访问量), `lastUpdated` (最后更新时间) |

### 2. 数据库权限设置

- **works**：所有人可读，仅管理员可写
- **categories**：所有人可读，仅管理员可写
- **admins**：仅管理员可读可写
- **visitStats**：所有人可读，仅管理员可写

## 五、云函数开发

### 1. 创建云函数

1. **创建云函数目录**：
   - 在项目根目录创建 `cloudfunctions` 文件夹
   - 在微信开发者工具中右键点击 `cloudfunctions` 文件夹，选择「同步云函数列表」

2. **创建云函数**：
   - 右键点击 `cloudfunctions` 文件夹，选择「新建 Node.js 云函数」
   - 输入函数名称，如 `login`、`addWork`、`getWorks` 等

### 2. 核心云函数

#### 1. 管理员登录 (`login`)

```javascript
// cloudfunctions/login/index.js
const cloud = require('wx-server-sdk')
cloud.init()
const db = cloud.database()

exports.main = async (event, context) => {
  const { username, password } = event
  
  try {
    const admin = await db.collection('admins').where({
      username: username
    }).get()
    
    if (admin.data.length === 0) {
      return {
        success: false,
        message: '用户名不存在'
      }
    }
    
    if (admin.data[0].password !== password) {
      return {
        success: false,
        message: '密码错误'
      }
    }
    
    return {
      success: true,
      data: admin.data[0]
    }
  } catch (error) {
    return {
      success: false,
      message: error.message
    }
  }
}
```

#### 2. 获取作品列表 (`getWorks`)

```javascript
// cloudfunctions/getWorks/index.js
const cloud = require('wx-server-sdk')
cloud.init()
const db = cloud.database()

exports.main = async (event, context) => {
  const { categoryId } = event
  
  try {
    let query = db.collection('works')
    
    if (categoryId) {
      query = query.where({ category: categoryId })
    }
    
    const works = await query.orderBy('createdAt', 'desc').get()
    
    return {
      success: true,
      data: works.data
    }
  } catch (error) {
    return {
      success: false,
      message: error.message
    }
  }
}
```

#### 3. 上传作品 (`addWork`)

```javascript
// cloudfunctions/addWork/index.js
const cloud = require('wx-server-sdk')
cloud.init()
const db = cloud.database()

exports.main = async (event, context) => {
  const { title, description, images, category, year, dominantColor } = event
  
  try {
    const result = await db.collection('works').add({
      data: {
        title,
        description,
        images,
        category,
        year,
        dominantColor,
        viewCount: 0,
        createdAt: new Date(),
        updatedAt: new Date()
      }
    })
    
    return {
      success: true,
      data: result
    }
  } catch (error) {
    return {
      success: false,
      message: error.message
    }
  }
}
```

#### 4. 增加浏览量 (`increaseViewCount`)

```javascript
// cloudfunctions/increaseViewCount/index.js
const cloud = require('wx-server-sdk')
cloud.init()
const db = cloud.database()

exports.main = async (event, context) => {
  const { workId } = event
  
  try {
    // 增加作品浏览量
    await db.collection('works').doc(workId).update({
      data: {
        viewCount: db.command.inc(1),
        updatedAt: new Date()
      }
    })
    
    // 更新总访问量
    await db.collection('visitStats').doc('total').update({
      data: {
        totalVisits: db.command.inc(1),
        lastUpdated: new Date()
      }
    })
    
    return {
      success: true
    }
  } catch (error) {
    return {
      success: false,
      message: error.message
    }
  }
}
```

## 六、云存储使用

### 1. 上传图片

```javascript
// 前端代码
wx.chooseImage({
  count: 9, // 最多选择9张图片
  sizeType: ['compressed'], // 压缩图片
  sourceType: ['album', 'camera'], // 从相册选择或拍照
  success: function(res) {
    const tempFilePaths = res.tempFilePaths
    const uploadTasks = []
    
    // 上传每张图片
    tempFilePaths.forEach((tempPath, index) => {
      const uploadTask = wx.cloud.uploadFile({
        cloudPath: 'works/' + Date.now() + '-' + index + '.jpg', // 云存储路径
        filePath: tempPath, // 临时文件路径
        success: function(res) {
          console.log('上传成功', res.fileID)
        },
        fail: function(err) {
          console.error('上传失败', err)
        }
      })
      uploadTasks.push(uploadTask)
    })
    
    // 等待所有上传完成
    Promise.all(uploadTasks).then(results => {
      const fileIDs = results.map(res => res.fileID)
      // 调用云函数添加作品
      wx.cloud.callFunction({
        name: 'addWork',
        data: {
          title: '作品标题',
          description: '作品描述',
          images: fileIDs,
          category: '分类ID',
          year: 2024,
          dominantColor: '#3b82f6'
        },
        success: function(res) {
          console.log('添加作品成功', res)
        }
      })
    })
  }
})
```

### 2. 调用云函数示例

```javascript
// 调用登录云函数
wx.cloud.callFunction({
  name: 'login',
  data: {
    username: 'admin',
    password: 'password'
  },
  success: function(res) {
    if (res.result.success) {
      console.log('登录成功', res.result.data)
      // 保存登录状态
      wx.setStorageSync('admin', res.result.data)
    } else {
      wx.showToast({
        title: res.result.message,
        icon: 'none'
      })
    }
  },
  fail: function(err) {
    console.error('调用失败', err)
  }
})
```

## 八、云开发控制台使用

### 1. 数据库管理

- **查看数据**：在云开发控制台的「数据库」 tab 中查看和编辑数据
- **索引管理**：为常用查询字段创建索引，提高查询性能
- **权限设置**：设置集合的读写权限

### 2. 存储管理

- **文件管理**：查看和管理上传的文件
- **存储配置**：设置存储桶的访问权限

### 3. 云函数管理

- **函数列表**：查看已部署的云函数
- **日志查询**：查看云函数的运行日志
- **监控指标**：查看云函数的调用次数和执行时间

### 4. 统计分析

- **使用量统计**：查看云开发资源的使用情况
- **性能监控**：监控云函数的执行性能

## 九、注意事项

### 1. 安全问题

- **密码存储**：建议对管理员密码进行加密存储
- **权限控制**：严格设置数据库和存储的访问权限
- **云函数鉴权**：在云函数中添加身份验证逻辑

### 2. 性能优化

- **图片处理**：上传前压缩图片，减少存储和带宽使用
- **数据缓存**：合理使用本地缓存，减少云函数调用
- **批量操作**：使用批量操作减少数据库请求次数

### 3. 配额管理

- **免费额度**：云开发提供基础免费额度，注意监控使用情况
- **资源限制**：了解云函数的执行时间限制（最长5秒）和内存限制
- **付费方案**：如果流量较大，可考虑升级到付费方案

## 十、常见问题

### 1. 云函数部署失败

- **检查依赖**：确保 `package.json` 中的依赖正确
- **网络问题**：确保网络连接正常
- **代码错误**：检查云函数代码是否有语法错误

### 2. 图片上传失败

- **文件大小**：确保图片大小不超过限制（默认50MB）
- **网络问题**：确保网络连接稳定
- **权限问题**：检查存储桶的权限设置

### 3. 数据库操作失败

- **权限问题**：检查集合的权限设置
- **字段类型**：确保字段类型正确
- **查询条件**：确保查询条件有效

## 十一、开发建议

1. **模块化开发**：将云函数按功能模块化，提高代码可维护性
2. **错误处理**：完善前端和云函数的错误处理
3. **测试**：在开发环境充分测试功能，确保上线后稳定运行
4. **监控**：使用云开发控制台的监控功能，及时发现问题
5. **文档**：为云函数和数据库结构建立文档，方便后续维护

## 十二、总结

微信云开发为个人作品集小程序提供了便捷的后端解决方案，无需搭建服务器即可实现数据库、存储和云函数等功能。通过合理设计数据库结构、编写云函数和使用云存储，可以快速构建一个功能完整的作品集小程序。遵循最佳实践，注意安全和性能问题，确保小程序的稳定运行和良好的用户体验。
